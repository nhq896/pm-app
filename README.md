## Chạy ứng dụng

### - Frontend

```bash
cp .env.example .env
npm install
npm run dev
```

### - Backend

```bash
cp .env.example .env
npm install
npm run dev
```

- chạy lệnh dưới để seed dữ liệu cho bảng roles và các dữ liệu test khác

```bash
npm run seed:all
```
